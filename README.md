# Project to try Symfony with Docker

## Features

* Users registration
* User auth with JWT-tokens
* Tokens refreshing
* Creating posts with weather and news data by queue
* View news list with pagination
* View detail news page with comments
* Creating comments
## Getting Started

1. If not already done, [install Docker Compose](https://docs.docker.com/compose/install/)
2. Create .env file with your tokens for services
4. Run `docker-compose up -d`
